## 🛠️ How to Run Any Script
To run any of the scripts locally on your machine, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the specific day's folder:
   ```bash
   cd 100-days-of-code-python/Day_01_to_10
   ```
3. Execute the Python script:
   ```bash
   python day01_problems.py
   ```
